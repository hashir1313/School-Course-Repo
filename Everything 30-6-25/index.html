Get Started

Starting with HTML!

HTML stands for Hyper Text Markup Language. It is used to give basic structure to a website. It is like the body of a car.

1. HTML Boilerplate

First things first, let's start with the basic structure boiler plate of HTML. You can get this boiler plate code by typing ! in the first line and then pressing the enter button in VS Code. Here is the boilerplate code:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">

</head>
<body>

</html>

2. Heading Tags

The heading tag is used to display Headings on a webpage. There are 6 heading tags from <h1> to <h6>. Here is an example code:

<h1>This is Heading 1</h1>
<h2>This is Heading 2</h2>
<h3>This is Heading 3</h3>
<h4>This is Heading 4</h4>
<h5>This is Heading 5</h5>
<h6>This is Heading 6</h6>

Here is the default output without styling:

This is Heading 1

This is Heading 2

This is Heading 3

This is Heading 4

This is Heading 5
This is Heading 6

3. Paragraph Tags

The paragraph tag is used to display paragraphs on a webpage. Here is an example code:

<p>This is an example paragraph.</p>

Here is the default output without styling:

This is an example paragraph.

4. Anchor Tags

The anchor tag is used to display links on a webpage. It has an attribute href which is the link that we want to redirect the user. Here is an example code:

<a href="https://google.com>Visit Google</a>

Here is the default output without styling:

Visit Google

Lets Talk about CSS!

CSS stands for Cascading Style Sheet. It is used to style webpages, like the paint of a car. We can apply styling with three ways.

  1. Inline CSS
  2. Internal CSS
  3. External CSS

1. Internal CSS

It is wrriten inside HTML tags. It has the highest priority between all ways of writing CSS meaning it will be applied even when an external stylesheet is connected. Here is an example Code:

<p style="color: red";>This is a red paragraph.</a>

Here is the default output without styling:

This is a red paragraph.
